Place static assets here to be served under /static.

Suggested layout for the Cesium model used by the 3D flythrough:

static/
  models/
    CesiumAir/
      Cesium_Air.glb

You can copy Cesium_Air.glb from the CesiumJS SampleData of a local Cesium release, or use any other .glb model and adjust the path in app.py accordingly.
